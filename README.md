# Airtime

Your digital headquarters. Track focus, analyze insights, and audit your time with Airtime.

## Features
- **Automatic Time Tracking:** Focuses on active domains while you work.
- **Privacy First:** All data is stored locally in your browser.
- **Insights Dashboard:** Visual charts and statistics to help you audit your time.
- **Manifest V3:** Built using the latest Chrome extension standards.

## Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the folder containing this code.

## Permissions
- `tabs`: To track current active URL.
- `storage`: To save your focus history locally.
- `idle`: To pause tracking when you are away.
- `alarms`: For periodic data synchronization.
